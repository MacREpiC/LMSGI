Listas

Pondré ejemplos en los que usaré y explicaré brevemente las etiquetas y atributos más importantes del PDF Vol. IV.

Listas no ordenadas

Etiqueta <ul> se utiliza para insertar etiquetas dentro de ella.

Etiqueta <li> sirve para insertar elementos dentro de la etiqueta <ul>.

Atributos de listas no ordenadas

Atributo type se pone detro del atributo <ul> o también <li> y se utiliza para especificar el tipo de viñeta.

Estos valores se utilizan dentro del atributo type:

  • Valor circle
  • Valor punto
  • Valor cuadrado

Listas ordenadas

Etiqueta <ol> se utiliza para insertar etiquetas dentro de ella.

Etiqueta <li> sirve para insertar elementos dentro de la etiqueta <ol>.

Atributos de listas ordenadas

Atributo type se pone detro del atributo <ol> o <li> y se utiliza para especificar el tipo de viñeta.

Estos valores se utilizan dentro del atributo type:

Listas Ordenadas tipo 1:

  1. Windows
  2. Mac-OS
  3. Linux
  4. Unix

Listas Ordenadas tipo a:

  1. Windows
  2. Mac-OS
  3. Linux
  4. Unix

Listas Ordenadas tipo i:

  1. Windows
  2. Mac-OS
  3. Linux
  4. Unix

Listas Ordenadas tipo A:

  1. Windows
  2. Mac-OS
  3. Linux
  4. Unix

Listas Ordenadas tipo I:

  1. Windows
  2. Mac-OS
  3. Linux
  4. Unix

Listas de definición

Etiqueta <dl> se utiliza para insertar etiquetas dentro de ella.

Etiqueta <dt> sirve para insertar elementos el elemento a definir dentro de la etiqueta <dl>.

Etiqueta <dd> sirve para definir el elemento dentro de la etiqueta <dt>.
