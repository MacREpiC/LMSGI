Enlaces, imagenes y formularios

Pondré ejemplos en los que usaré y explicaré brevemente las etiquetas y atributos más importantes del PDF Vol. V.

Enlaces

Atributo <href> se utiliza para poner URL internas hacia archivos HTML o hacia sitios web.

<a href=https://google.com>

Etiqueta <form> se utiliza para indicar que a continuación vas a hacer un formulario. Dentro de ella se pueden poner atributos como action="emailto:pepe@gmail.com".

<a name="arriba"></a>

Crea un marcador en la página con el nombre "arriba". No tiene texto visible ni contenido visible para el usuario.

<a href="arriba">Quiero ir arriba!!!</a>

Este es un enlace que incluye texto visible para el usuario ("Quiero ir arriba!!"). El atributo href está configurado con el valor #arriba para mandar al usuario hacia arriba.

Formularios

Etiqueta <fieldset> es utilizada para agrupar elementos relacionados en un formulario dentro de un cuadro. Se utiliza para estructurar y organizar formularios, ayudando a mejorar la accesibilidad y la legibilidad del código.

Etiqueta <legend> Esta etiqueta se utiliza dentro de un elemento <fieldset>. Define una leyenda o título descriptivo para el grupo de elementos relacionados que están dentro del <fieldset>. Proporciona una descripción o contexto para los elementos agrupados.

Etiqueta <botton>se utiliza para crear un botón en una página web. Puede contener texto, imágenes u otros elementos HTML anidados dentro de ella. Esta puede tener los siguientes atributos:

  • type Define el tipo de botón. En este caso, cuando se establece como "button", indica que se trata de un botón estándar. Otros valores posibles para este atributo incluyen "submit" para enviar formularios y "reset" para restablecer los campos de un formulario.
  • name Nombre del botón (se utiliza al enviar formularios).
  • value Valor asociado al botón (se utiliza al enviar formularios).

Los botones se pueden crear de la siguiente foma:

<input type="button" value="Haz clic aquí">

Imágenes

Para insertar una imagen en HTML, se utiliza la etiqueta <img>. El atributo src especifica la URL o la ruta de la imagen.

Para configurar el ancho y el alto de la imagen, se utilizan los atributos width y height. Estos atributos pueden establecerse en píxeles (px) o porcentajes (%).

<img src="ruta_de_la_imagen.jpg" alt="Descripción de la imagen" width="300" height="200">
