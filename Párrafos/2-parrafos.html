Párrafos

Pondré ejemplos en los que usaré y explicaré brevemente las etiquetas y atributos más importantes del PDF Vol. III.

Etiquetas

Etiqueta <p> sirve para crear párrafos

Etiqueta <pre> sirve para mostrar el texto preformateado

Etiqueta <br> sirve para hacer
saltos de
línea

Etiqueta <!---comentario---> sirve para añadir comentarios que no estará visibles a la hora de representar el archivo HTML en el navegador

Etiqueta <em> marca el texto indicando que su importancia es mayor, poniendo este en cursiva

Etiqueta <strong>marca el texto determinado con mayor importancia, poniendo este en negrita

Etiqueta <b> pone el texto en negrita

Etiqueta <i> pone el texto en cursiva

Etiqueta <strike> pone el texto tachado

Etiqueta <big> aumenta el tamaño del texto

Etiqueta <small> disminuye el tamaño del texto

Etiqueta <sub> pone el texto como subíndice

Etiqueta <sup> pone el texto como superíndice

Etiqueta <u> pone el texto subrayado

Etiqueta <font> podemos insertar el texto entre las etiquetas para luego añadirle atributos como cambiarle el color a la letras, el tamaño, la tipografía, etc.
